## Quick Installation

    git clone https://github.com/cattallin/laravel-automtion-test.git laravel-automation-test

    cd laravel-automation-test

    composer install

    php artisan migrate

    php artisan serve


    Run tests command:

    php vendor/bin/codecept run

    File with test:
    /tests/functinal/TaskCest.php
